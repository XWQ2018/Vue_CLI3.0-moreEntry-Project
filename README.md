# moreentryproject

## 多个项目入口，单个项目代码打包

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your end-to-end tests
```
yarn run test:e2e
```
### index页打包命令
```
yarn run build_index
```
### page2页打包命令
```
yarn run build_page2
```
### 综合打包
```
yarn run build_allProject
```
